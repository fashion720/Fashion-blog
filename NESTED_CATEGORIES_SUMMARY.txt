🔄 HOW TO USE
────────────────────────────────────────────────────────────────────────────

STEP 1: Create Main Categories
  Location: http://localhost:3000/keystatic
  - Categories collection
  - Create category: "TRENDS"
  - Leave "Parent Category" empty
  - Save

STEP 2: Create Sub-Categories
  Location: Same (Keystatic admin)
  - Create category: "SPRING"
  - Set "Parent Category" to "TRENDS"
  - Save
  - Repeat for SUMMER, AUTUMN, WINTER

STEP 3: Assign Posts
  When creating a post:
  - Select Categories field
  - Assign to "TRENDS" (main category)
  - AND/OR assign to "TRENDS-SPRING" (sub-category)
  - Save

STEP 4: View Navigation
  - Desktop: Hover over "TRENDS" to see dropdown
  - Mobile: Tap menu, see indented categories
  - Both show smooth animations
